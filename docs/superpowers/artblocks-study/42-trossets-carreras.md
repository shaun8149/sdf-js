# 第四十二课: Trossets — Anna Carreras

- **ArtBlocks #147** · p5 纯 2D · 12KB · CC BY-NC-SA 4.0 → recipe-only (SA 从严)
- 视觉: 网格上的线缠绕小块 (trosset = 加泰兰语"小片"), 松弛童趣

## 分流判定: 2D-core, 文档课 (画家字典架构已入库)

16 个具名 trosset 画家 (trossetL/R/H/V/P/X/M/VV/XX/LL/PP/MN/MS…)
按格子抽签 — **画家字典第二目击** (L17 Cargo 首见), 但比 Cargo 多
一层: 字典键有方向语义 (L/R/H/V = 左/右/横/竖), 邻格衔接靠方向
命名约定成立, 缠线能跨格连续。

## 值得带走的 idiom

1. **字典键携带拓扑语义**: 画家名即接口 (L 出口在左) — 抽签时按
   邻格已定画家过滤 possibles, 字典 + 约束传播 = 波函数坍缩的
   手写小版。cargo-dashes v2 若做"跨块连续纹理"可采。
2. **地名调色板**: cMontseny/cSalines/cIbiza/cAltafulla/cOlivos —
   调色板以真实地方命名, 5 色一组全部手调。具名调色板共识的
   人文变体: 名字不是标签是记忆坐标。
3. Carreras 的松弛感来源: 缠线端点带 ±抖动, 网格严格但笔触不严格
   — 秩序在结构层, 松弛在笔触层, 两层分开调。

## 一句话学到的

字典键可以是接口约定: 名字携带拓扑, 抽签自动变成约束传播。

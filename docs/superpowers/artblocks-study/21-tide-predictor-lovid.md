# 第二十一课: Tide Predictor — LoVid

- **ArtBlocks #376** · p5 纯 2D · 4.6KB (全批最小) · **CC BY-NC-ND 4.0 → 严格 recipe-only**
- 视觉: 模拟视频合成器式 RGB 拍频条纹, 同步滑移的对角漂移

## 分流判定: 2D-core (逐像素扫描光栅) → 2D 队列, 出家族

LoVid 是模拟视频艺术双人组; 这件作品把他们的 video-synth 实践翻译成
一个 1000×1000 的逐像素扫描循环 — 没有几何、没有噪声、没有对象,
只有**一根 1D 扫描线和三条三角波**。

## 解剖 (recipe)

1. **扫描序即空间**: 唯一的坐标是扫描索引 i (0..1e6), 折行回 2D 后,
   周期接近行宽 → 竖条纹; 差几个像素 → 对角漂移 (CRT 失同步的斜纹)。
   空间结构是"折行"这个操作的伪影 — 一维时序艺术免费获得二维构图。
2. **三通道独立三角波**: R/G/B 各自 `255-|255-(i%p)·k|`, 周期互异 →
   颜色是三条波的拍频, 从不显式计算"颜色"。
3. **周期随机游走 + 概率再同步**: 波谷处 0.01% 概率换周期、1% 概率
   增益微调; 每帧一掷 sync_probs (0.01/0.05/0.1/0.5, 铸造决定) 全体
   归位 — 滑移与锁定的呼吸节奏就是作品的"潮汐"。
4. drawWait 1ms→5000ms→500ms 的节拍器: 首帧即像, 之后慢演化。

## Port: scan-tides 家族 (registry.js, DECOR_V=1 下新增)

调色板约束改写: 三 RGB 通道 → 两主题色的 OKLab 三角波混合 (通道 A) +
失谐第二周期的峰值刻线 (通道 B, 拍频痕)。扫描游标跨行连续 (折行伪影
保留), 周期逐行随机游走 + 概率再同步。人格 = 周期长短 × 游走幅度 ×
再同步概率。ND license: 零代码复制, 全部按 idiom 重写。

## 一句话学到的

一维时序折成二维就是构图 — 周期失谐给纹理, 再同步给节奏, 都不需要几何。

## 解剖 (recipe)

1. **加权 trait 表**: `[["Sweep",68],["Focal",11],["Offset",12],
   ["Vertical",6],["Flock",3]]` — 构图模式带百分比权重, 68% 常规
   3% 珍品, 权重就是发行计划。
2. **调色板色也带权重**: 每板每色 `[hex, weight]` (含 weight 0 =
   只做点缀不做主色) + 每板专属 textureC 纹理色 — 偏置不禁止。
3. **纸片形 = Verlet 软体, 主动拒绝毛边**: 纸片是圆环粒子 + 弹簧
   约束的软体 (内注 credit David Lu), morph 松弛成形; warp 用**单尺度**
   噪声场位移; `detectJagged()` (段长过阈判毛糙) **主动拒绝**尖锐边 —
   毛了就重试直到平滑。一读写的"双尺度噪声 (慢起伏+细毛刺) + 偶发
   尖锐内切撕口"机制**不存在**。"撕纸感"的真实来源: 图案纹理填充
   (waves/flowers/hexagons/vines 等 8 种 pattern generator) + 泼溅
   (4 万个微三角/微矩形纤维) + 颗粒 + 噪声纸边框。
4. 双 PRNG 交替 + 1e6 预热**第四目击** — 共识铁证, 就此停止计数。
   加 100 次额外空转 (`for e<=100 random_dec()`) — 双保险预热。
   另档: 按颜色数补齐 PRNG 调用次数到定额 (调用数归一化)。

## Port: torn-paper 家族 (registry.js, DECOR_V=1 下新增)

patch = 46 点径向环, 半径吃双尺度噪声 + 概率撕口 (相邻 3 点半径
×0.55), squash+旋转防正圆; 填充 + 毛边描线; **accent 三倍权重抽色**
(Xie 加权板的最小实现)。人格 = 片数 × 撕口概率 × 粗糙度。
适配 organic/hr。
**归因更正**: 整套撕边 recipe (双尺度噪声 + 撕口内切) 是我们的发明,
不是"Xie 机制的最小实现" — 原作是 Verlet 软体且 detectJagged 主动
拒绝尖锐边 (见解剖 3); 原作的招牌"图案纸片" (纹理填充) 家族也未做。
装饰效果自立, 但系谱叙述需按此更正 — 家族二修待 DECOR_V 纪律裁决。

## 一句话学到的

权重是比开关更文明的控制: 不禁止任何颜色, 只是让某些颜色更常被想起。

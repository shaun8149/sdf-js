## 解剖 (recipe)

1. **加权 trait 表**: `[["Sweep",68],["Focal",11],["Offset",12],
   ["Vertical",6],["Flock",3]]` — 构图模式带百分比权重, 68% 常规
   3% 珍品, 权重就是发行计划。
2. **调色板色也带权重**: 每板每色 `[hex, weight]` (含 weight 0 =
   只做点缀不做主色) + 每板专属 textureC 纹理色 — 偏置不禁止。
3. **撕纸边 = 双尺度噪声**: 慢起伏 (纸片形) + 细毛刺 (deckle 纤维),
   偶发尖锐内切 (撕口)。
4. 双 PRNG 交替 + 1e6 预热**第四目击** — 共识铁证, 就此停止计数。
   加 100 次额外空转 (`for e<=100 random_dec()`) — 双保险预热。

## Port: torn-paper 家族 (registry.js, DECOR_V=1 下新增)

patch = 46 点径向环, 半径吃双尺度噪声 + 概率撕口 (相邻 3 点半径
×0.55), squash+旋转防正圆; 填充 + 毛边描线; **accent 三倍权重抽色**
(Xie 加权板的最小实现)。人格 = 片数 × 撕口概率 × 粗糙度。
适配 organic/hr。

## 一句话学到的

权重是比开关更文明的控制: 不禁止任何颜色, 只是让某些颜色更常被想起。

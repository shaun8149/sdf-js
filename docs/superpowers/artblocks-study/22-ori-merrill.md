## 分流判定: 2D-core (纯几何引擎) → 2D 队列, 出家族

无 shader、无噪声贴图 — 核心是一套**折纸几何引擎**: 参数化线段相交
(带 t/s 双参数)、点在多边形内测试、多边形沿折线剖分。webpack 模块化
(生成艺术里少见的工程化打包, Merrill 是工程师型作者)。

## 解剖 (recipe)

1. **折叠 = 线剖多边形**: 顶点按折线侧别分类, 跨侧的边插入交点,
   两侧各出一个子多边形 — 没有任何 3D, "折"是平面拓扑操作。
2. **深度着色卖出立体感**: facet 按折叠深度渐变色调 — 眼睛把色阶
   读成纸的朝向 (与 L19 Screens 的 facet 亮度同一招, 几何版)。
3. **永远剖最大 facet**: 构图平衡的秘密 — 不挑最大就会出现一角碎骨
   一角空白。贪心面积 = 免费的构图均衡器。
4. 折轴偏爱轴对齐 (±jitter): 全随机角度出乱纸团, 轴对齐基调出折纸感。

## Port: paper-folds 家族 (registry.js, DECOR_V=1 下新增)

区域矩形当纸, K 次"剖最大 facet", OKLab 深度渐变 + 折痕描线。
人格 = 剖分次数 × 折轴抖动 × 色调展幅。适配 organic/hr (软拼贴感)。
与 shard-mesh (碎三角) / block-mosaic (网格) 的差异: **少而大** 的 facet。

## 一句话学到的

"折"不需要 3D — 剖分给形, 深度给色, 贪心最大给平衡。

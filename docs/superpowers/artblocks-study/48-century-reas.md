## 值得带走的 idiom

1. **janky 开关**: `let janky = true` — 刻意的不完美是一个具名
   布尔 trait: 切片边缘的抖动/错位受它统一控制。"手感"被工程化
   为可铸造的开关 (L23 图层开关组的美学版)。
2. 双调色板配对: colorsA (面) + lineColorsA (线) 分池 — 面色与
   线色各自策展, 同一构图两个色彩预算 (角色分工共识的又一例)。
3. maxSlices=50 + sliceOrder 洗牌: 构图 = 有序切片的一个排列 —
   与 L38 Ringers "构图即排列"同构, 一维版。
4. 项目编号 #100 = "CENTURY" — 编号本身进入作品概念, 平台元数据
   作为语义层 (与 L29 Quine 的自反性同谱系)。

## 一句话学到的

不完美也要工程化: janky 是一个开关, 不是一堆散落的随机抖动。

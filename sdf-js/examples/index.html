sdf-js — examples

分两层：底层 sdf/ = 形状本身（primitives + ops + 链式 API）， 上层 ca/ = 基于 SDF 的离散结构生成器（form × generator 解耦）。

sdf/ — 形状层

SDF primitives、布尔、变换、shader 编译，以及在它们之上的渲染展示。

  • 2d.html 2D 距离场可视化（蓝/红色带 + 边界等值线），所有 2D 原语在场
  • 3d.html 3D raymarching 网格，纯 JS 单线程，每个面板一个链式表达式
  • editor.html SDF 编辑器 — 拖参数实时看图，输出 import 代码
  • cactus.html BOB 仙人掌场景 — 像素剪影版（JS）
  • cactus-painted.html 同场景的笔触版（p5，BOB 风格点画）
  • cactus-shader.html 同场景的 fragment shader 版（更快更高清）
  • cactus-sand-shader.html 仙人掌沙画版（shader）
  • painted-scenes.html BOB 6 个场景随机笔触版
  • scenes-debug.html 所有场景的 silhouette debug 视图
  • scenes-sand.html 所有场景的沙画版（含鸟）

ca/ — 元胞自动机层

kjetil golid ApparatusGenerator 的端口，把 SDF 当作 isInside 谓词消费 — 任何形状都能生成同款拼合纹理。SDF × discrete generator 矩阵的第一格。

  • ca.html SDF + CA（机器人填充） — 静态版，参数面板齐全
  • ca-animate.html 同款机器人的 scatter → assemble 装配动画

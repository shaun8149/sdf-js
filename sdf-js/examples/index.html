Generative work, in signed distance fields.

The playground behind Atlas — the generative renderer BOB, a shelf of art-style renders, and LLM-written sketches, all built on the same form language: (x, y) → number. Not the product; the play.

BOB & generative

BOB — the generative GPU renderer — and the systems that feed it. Same seed, same world, shareable by URL.

  • autoscope · BOB BOB GPU across six scene templates — city / sea / forest / village / city-axis / abstract. Autoscope-style: same hash → same scene → a link you can share.
  • scenedata-demo Paste SceneData JSON → compile → BOB GPU. Sphere/ground, repetition + animation, and a full autoscope-style scene (camera dolly, waves, hue-shifted shadow).
  • shader-lambert-browser Fly through eight preset SDFs — WASD pointer-lock, compiled to WebGL Lambert at 60fps.

Renderers & art styles

One distance field, many hands. The same SDFs drawn as silhouette, brushstroke, sand, hatching, and shader.

  • render-showcase Four renderers — silhouette, bands, sand, painted — over the same SDFs. The overview plate.
  • painted-scenes Sixteen scenes as gridded brushstrokes, in the lineage of Tyler Hobbs and kjetil golid.
  • streamline-scenes Contour-following hatching across fourteen scenes.
  • scenes-sand Sand-painting: cumulative random sampling, three-tier colouring.
  • pasma-capsules Surface-aware 3D rayhatching on SDF capsules — intensity-modulated line spacing.
  • cactus-shader The cactus scene compiled to a single GLSL fragment shader.
  • cactus-sand-shader Sand-painting, done entirely in the shader.
  • torso A back-view torso in polygons — toggle outline against streamline hatch.

LLM × SDF

A language model, asked for a short noun phrase, writing the scene as code. Some land, some break beautifully.

  • text → SDF Type a prompt → Claude writes JS SDF code → live render. Six renderers × five patterns, 2D/3D, SVG export.
  • tree Round two — a tree, written from a two-word prompt.
  • cathedral Round two — arches and light.
  • butterfly Round two — symmetry from a single word.
  • seurat Round two — pointillism, in distance fields.
  • round 1 The first pass — seven scenes, bugs kept as they were, as essay material.

Fields, cellular, tiles

The other form families — Perlin noise, cellular automata, plotter tiles — in the lineage of Harvey Rayner and Vera Molnár.

  • ca kjetil golid's Apparatus, running on any SDF.
  • ca-animate The same grid, scattered then assembled.
  • noise-curves A Perlin flow field, densely packed with streamlines.
  • alice Proto field × line-clipped tile fill × generative palette, oscillating through a stereographic projection.
  • 2d The raw distance field — blue inside, red outside, banded.
  • 3d Pure-JS raymarching, one chainable expression per panel.

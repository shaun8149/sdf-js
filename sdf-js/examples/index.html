sdf-js source

Composable signed distance functions.

A JavaScript port and extension of fogleman/sdf. Form, generator, render — orthogonal layers meeting through (x, y) → number. SDF is one such field; Perlin noise and proto field are siblings.

Form

SDF primitives, booleans, transforms; the algebra everything else consumes. Sibling field families live in field/ (Perlin noise, proto, combinators).

  • render-showcase Four renderers — silhouette, bands, sand, painted — over the same SDFs.
  • 2d Distance-field visualisation: blue inside, red outside, periodic bands, zero-isoline.
  • 3d Pure-JS raymarching grid; one chainable expression per panel.
  • mvp · text → SDF Type a prompt → Anthropic Claude writes JS SDF code → live render. 4 renderers × 5 patterns × 2D/3D selectable. SVG export. The flagship LLM-native creative app.
  • painted-scenes 16 scenes rendered as gridded brushstrokes (Tyler Hobbs / kjetil-style). 2D + 3D via probe.
  • streamline-scenes Contour-following hatching (2D analog of Pasma rayhatching) across 14 scenes.
  • scenes-sand Same scenes as sand-painting: cumulative random sampling, three-tier colouring.
  • scenes-debug Silhouette debug view across all scenes; useful when authoring new SDFs.
  • llm-round1 LLM × SDF round 1 (minimal-context prompt): 7 scenes, bugs preserved as essay material.
  • test-torso Back-view female torso polygon. Toggle outline (shell+intersection) vs hatch (streamline).
  • cactus-shader The cactus scene as a fragment shader — same SDF, GLSL compilation path.
  • cactus-sand-shader Sand-painting style implemented entirely in a shader.

Generator

Discrete structure generators that consume an SDF as an isInside predicate.

  • ca Cellular automaton (kjetil golid's Apparatus) running on any SDF; static.
  • ca-animate Scatter-then-assemble animation of the same CA grid.

Flow

Streamline generators on scalar/vector fields. field as the form layer, streamline as the path generator, render.flowLines as the renderer.

  • noise-curves Perlin noise flow field + dense-packed streamlines. Phase A minimum pipeline.

Tile

Halftone / plotter paradigm: regular grid, each cell drawn from field samples. Lineage of Harvey Rayner, Vera Molnar.

  • alice Proto field × line-clipped tile fill × generative palette. Stereographic-projection oscillation.

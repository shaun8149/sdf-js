永不重复的外衣

Atlas 的每一份 deck 在生成时铸造一个 64 位随机 hash — 修饰纹理的每个决策都由它派生。 持有 hash 的作者可以永久复现这一件; 不知道 hash 的任何人, 用同样的文本和同样的工具, 也铸不出同一件。 这不是水印, 是生成艺术的 provenance。

最左是"裸"的一页; 右边四张是同一页内容的四次铸造 — 家族、纹理、性格全部由各自的 hash 决定。
